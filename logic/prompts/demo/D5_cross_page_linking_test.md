# D5 — Cross-page linking test

## Purpose

Check whether the vault behaves like a connected wiki rather than isolated notes.

## Prompt body

```text
Trace the knowledge path for [PRODUCT] through this vault.

Show:
1. source notes
2. linked product page
3. linked pricing page if any
4. linked term pages
5. what knowledge is current
6. what knowledge is still incomplete

I want to see whether this vault behaves like a connected wiki rather than isolated notes.
```
