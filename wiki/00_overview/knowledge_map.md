# Knowledge Map

## 中文标题
知识地图

## Purpose
This page shows how major knowledge domains connect.

## Domain Map

### 1. Product Information
Contains:
- product names
- ingredients
- dosage
- NPN
- pack size
- suggested use
- recommended use
- target users

Lives in:
- `wiki/02_products/`

### 2. Material Information
Contains:
- dosage forms
- packaging forms
- oil base
- chelated mineral forms
- material-related product structure

Lives in:
- `wiki/03_materials/`

### 3. Marketing Messaging
Contains:
- selling points
- brand positioning
- social proof
- event references
- endorsements
- approved vs risky claims

Lives in:
- `wiki/04_marketing/`

### 4. Translation and Terminology
Contains:
- canonical English terms
- preferred Chinese translations
- aliases
- discouraged variants
- EN/ZH usage guidance

Lives in:
- `wiki/05_translation/`
- `wiki/09_term_registry/`

### 5. Pricing
Contains:
- price evidence
- channel differences
- retail vs livestream vs promo distinctions
- pricing guardrails

Lives in:
- `wiki/06_pricing/`

### 6. Inquiry Patterns
Contains:
- common customer asks
- reply logic
- missing-info patterns
- safer customer-facing templates

Lives in:
- `wiki/07_inquiry_patterns/`
- `wiki/08_reply_templates/`

## Flow Map
Raw source file
-> Source Note
-> Candidate claims
-> Compare with existing knowledge
-> Update wiki pages
-> Archive superseded knowledge
-> Record in log
-> Surface in index
